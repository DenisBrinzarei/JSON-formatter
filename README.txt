This project was created in IntelijiIdea, using Java11, and Maven dependencies on GSON library.
To run the code, clone the repo and execute "JSON-formatter/src/main/java/JsonFlattener.java/main(String[] args)" function with valid JSON as an argument.
Alternatevly, run the "JSON-formatter/src/main/java/JsonFlattener.java/JsonFlattenerTest()".
